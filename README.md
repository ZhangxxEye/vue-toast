# vue-toast
webpack构建的vue组件开发模板
## install
NPM
```
npm install @nat/vue-toast
```
## 使用
```
import toast from '@51npm-test/vue-toast'
Vue.use(toast)
this.$toast('出错了.....');
```
## 启动
```
npm run dev
```
## 编译
```
npm run build-lib
```
## 开发环境调试
docs/pages目录中的test.vue文件中引入组件，启动本地服务调试
```
npm run dev
```
